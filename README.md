# data-file-converter
An application made with PySide6 to convert files between json, yaml and xml formats, unit tested with pytest.

- Can convert selected files or files found in a selected, with the option to include subfolders and specific file extensions.
- Can place converted files to a selected folder or replace the original files, in which case there is an option to make backups to a selected folder.
- Output of operations are written out to a textbox to inform the user.
